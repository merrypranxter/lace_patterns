# Reticella

## Definition

Reticella is early needle lace evolved from cutwork, using rigid geometric grids, squares, circles, and radiating forms.

## Visual signature

- identifiable thread / void structure
- repeated motif logic
- controlled openness
- visible construction method
- edges and joins that reveal how the lace is made

## Common motifs

- flowers
- leaves
- scrolls
- medallions
- diamonds
- fans
- scalloped edges
- picots
- mesh grounds

## Prompt language

Use phrases such as:

```text
Reticella, early needle lace evolved from cutwork, using rigid geometric grids, squares, circles, and radiating forms, delicate openwork, visible thread paths, active negative space, motif-and-ground hierarchy, transparent holes casting tiny shadows, textile close-up, high-detail lace structure
```

## Shader / procedural translation

Represent this lace as:

- thread mask over transparent void
- graph nodes for intersections
- bezier curves for thread paths
- repeated cells for ground
- thicker outlines for corded motifs
- small dots or loops for picots
- alpha channel for holes

## Anti-drift

Do not reduce this to generic floral fabric. The openwork and construction logic must remain visible.
