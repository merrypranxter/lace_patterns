# Lighting Templates

## Modules

- backlit
- raking light
- macro shadow
- soft diffuse
- high contrast silhouette

## Use pattern

```text
[base subject] + [specific lace family] + [motif module] + [ground module] + [material module] + [lighting module] + [anti-drift clause]
```

## Anti-drift clause

```text
real openwork lace, visible thread paths, transparent holes, structural negative space, not a flat printed lace texture
```
