Create [SUBJECT / SURFACE / GARMENT / PATTERN] using [SPECIFIC LACE FAMILY].
The lace should be constructed from [THREAD STRUCTURE], with [MOTIF SET] arranged through [REPEAT / RADIAL / BORDER / ALLOVER LOGIC].
