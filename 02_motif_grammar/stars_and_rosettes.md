# Stars And Rosettes

## Role In Lace

radial lace motifs with points, rings, and symmetry.

This motif family helps define the rhythm of lace by controlling where thread gathers, where voids open, and how the eye travels across the textile.

## Visual traits

- thread and negative-space balance
- repeating sub-units
- visible loop or crossing logic
- dense-to-open transitions
- compatibility with borders, fields, and medallions

## Procedural knobs

- scale
- repetition count
- thread thickness
- hole size
- motif spacing
- symmetry mode
- edge decoration
- noise / hand irregularity

## Prompt fragment

```text
stars and rosettes, delicate thread paths, open negative space, visible lace construction, transparent holes, tiny cast shadows, motif hierarchy
```
