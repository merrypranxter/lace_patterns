# Thread Thickness

Thin thread, heavy cord, raised outlines, gimp, and chunky crochet scale.

## Things to specify

- thread color
- backing color or transparency
- scale of holes
- thread thickness
- motif density
- edge type
- lighting direction
- material sheen
- handmade irregularity vs machine precision

## Good visual language

```text
delicate openwork, thread-level detail, transparent voids, scalloped edge, tiny picots, floral motif hierarchy, mesh ground, soft cast shadows, visible raised cords
```

## Bad visual language

```text
generic lace texture, white pattern, decorative fabric
```
