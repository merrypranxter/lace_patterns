# Lace Patterns Expansion Pack 01

This expansion pack turns `lace_patterns` into a much deeper lace-pattern style and structure repo.

It treats lace as:

- negative-space textile architecture
- looped, knotted, twisted, braided, cut, drawn, or embroidered openwork
- graph grammar
- motif field
- repeat system
- transparency mask
- shader / generative art structure
- promptable visual language

Lace is not just "pretty white floral fabric." Lace is a family of openwork systems built from thread paths, voids, anchors, crossings, picots, nets, bars, loops, motifs, scallops, and repeats.

Use this pack for:

- AI art prompting
- textile pattern generation
- shader masks
- SVG / vector ornament
- laser-cut or screenprint translations
- crochet, tatting, bobbin, needle, and machine-lace references
- generative art based on thread graphs and negative space

## Drop-in instructions

Unzip this pack into the root of the repo.

It is intentionally additive. It does not overwrite an existing README unless you choose to merge content manually.
