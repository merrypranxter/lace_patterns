# Picot Construction

Picots create tiny looped teeth along edges and rings.

## Construction behavior

- creates local density or openness
- changes edge silhouette
- defines motif rhythm
- affects shadow and transparency
- can be converted into a procedural mark

## Prompt fragment

```text
visible picot construction, thread-level construction detail, delicate openwork, transparent voids, tactile textile structure
```

## Code translation

This unit can be represented as a curve, loop, small circle, repeated arc, filled grid cell, node cluster, or decorative edge marker.
