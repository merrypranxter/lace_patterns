# Lace As Negative-Space Architecture

Lace behaves like architecture made from thread.

The thread is the structure. The holes are rooms, windows, channels, and courtyards.

## Important architectural traits

- **Load paths**: where thread carries visual and physical structure.
- **Open cells**: repeated holes that make a ground.
- **Bridges**: bars or brides connecting motifs.
- **Anchors**: knots, loops, picots, or stitches that hold intersections.
- **Borders**: scallops, picot edges, bands, hems, insertions.
- **Motif islands**: flowers, leaves, medallions, scrolls, diamonds.
- **Void hierarchy**: large, medium, and small holes create visual depth.

## Generative translation

Represent lace as a graph:

- nodes = knots / crossings / anchors
- edges = thread paths
- faces = holes
- weights = thread thickness / tension
- masks = solid thread vs transparent void
